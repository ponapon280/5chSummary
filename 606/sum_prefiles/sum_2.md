   - **問題**: GIF変換でサイズオーバー（400x300/6MB制限）。他ユーザーは大容量動画投稿可能。
   - **解決策**:
     | 方法 | 詳細 | レス番号 |

     | FFmpegコマンド例 | `ffmpeg -i input.mp4 -start_number 0 -vf scale=1280:-1:flags=lanczos -r 32 -q:v 1 ugoira/%%03d.jpg`<br>（scale=横幅調整、-r=FPS、q:v=圧縮率。枚数多→-r半分でガクチカ対応） | >>253 |
     | WebP分解 | 間引/減色で枚数/サイズ調整（>>249）。 | >>249 |
     | フレーム時間 | (1/FPS)*1000 msで設定（小数切り捨て） | >>253 |
   - **感想**: GIF非推奨（メタ不明でイライラ）。お絵描きサイトらしい運用（>>250）。
