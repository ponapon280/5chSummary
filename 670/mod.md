# 元の本文
**人気モデルのまとめ（ログからの推測）**  
ログ全体から、現在最も積極的に話題に上がり、主力として使用・比較されているのは **anima** と **Krea2** の2モデルです。animaは「ワークフローのシンプルさ」「即時的な画質の良さ」「プロンプト理解度の高さ」「LoRA学習の速さ」「エロ/NSFW表現の強み」などで高評価され、日常生成や二次画像の主力として繰り返し挙げられています。Krea2は「自然言語プロンプトへの反応性・理解度の高さ」「表情やオリジナルキャラの表現力」「手・構図の精度」「将来的な主流化の可能性」で評価され、特にプロンプト制御や二次絵・実写寄り用途で人気です。  

illustrious（リアス/IL）は過去の主力モデルとして言及が多く、「絵師タグの混ぜやすさ」や画風LoRA作成用途で残っていますが、性能面でanimaなどに移行する動きが目立ちます。動画生成ではWan（Wan2.2）とLTXが現役最前線として採用されており、NovelAIやQwen-Imageは特定用途（背景強化、画像編集）で散見されます。SDXLは「レガシー」扱いながらアニメイラスト特化で一部残存。全体として、anima/Krea2の二強状態で、用途による使い分け（anima：タグ打ち・特定キャラ安定、Krea2：オリジナル性・自然言語）が明確です。[[1]](https://www.krea.ai/)[[2]](https://huggingface.co/krea)

## 各モデルの詳細と選ばれている理由（テキストベース）
- **anima（Anima）**  
  言及が非常に多く、ログの中心。選ばれる主な理由は、ワークフローがシンプル（余計なノード不要で綺麗に出る）、プロンプト理解度・制御性が高く「望み通りに出てくる」、画質の即時安定性、エロ/特殊性癖表現の強さ、LoRA学習が高速（リアス比で大幅短縮）、方向指定などの構図制御が優れている点。ホラー表現や3D素材からの安定出力も評価。Krea2との併用や住み分けが議論される。

- **Krea2（Krea / Krea2 / redcraft kreaなど派生）**  
  複数のログで最も言及が多いモデル。選ばれる理由は、プロンプト反応性・自然言語対応の高さ、表情・手・構図の精度、オリジナルキャラやdanbooruにない概念の表現力、二次絵への対応力。将来的にVRAM環境向上で主流化の可能性、LoRA学習・edit機能の進化も評価。一方でNSFWの過激さやLoRA学習時間の長さが課題として挙げられる。animaとの「ジャンクフードとレストラン」的な棲み分けが指摘される。

- **illustrious（イラストリアス / リアス / IL）**  
  過去の主力。選ばれる理由は、絵師タグの自然な混ぜやすさ・ブレンド性能の高さ、画風LoRA作成用データセット生成での有用性、クオリティタグとの相性。ただしanimaなどに性能で押され、日常使いは移行傾向。線質（細め）で比較されることも。

- **Wan（Wan2.2 / WAN2）**  
  動画生成用途で現役最前線。選ばれる理由は、24fps対応、リファレンス機能、実用性（ループしないなど）。静止画比で使用頻度が落ちる声もあるが、ローカル動画生成の選択肢として採用。

- **Qwen-Image（Qwen-Image / QIE / qwen-image-edit / qwen3.8）**  
  画像編集用途で言及。選ばれる理由は、編集タスクでの実用性と高クオリティ（Fableクラス以上と評価）。西側モデルを上回る可能性が指摘される。

- **その他**  
  - NovelAI（NAI / NAI5）：背景・無機物描写の強化で言及。話題自体は少ない。  
  - LTX（LTX-2.3）：動画生成で生成速度の速さが評価され、Wanと併用。  
  - Z-Image：三次画像用途でKrea2と併用。  
  - FLUX：過去の比較対象として散発的。  
  - SDXL：アニメイラストの衣装再現性などで未だ優位だが「終わった」扱いが多く、移行が進む。

**傾向**：静止画はanima/Krea2中心でプロンプト制御・画質・NSFWが鍵。動画はWan/LTX。illustriousは画風混ぜ用途で残存。LoRA作成速度や制御性が移行の大きな要因。

## Web検索による参考情報
- **Krea2**: Krea.aiが開発した独自のファンデーションモデル（Krea 2 Raw/Turbo）。美学重視で、ComfyUIなどでクリエイティブ画像生成に強く、LoRA対応も進む。オープンソース版も提供されており、2025-2026頃に注目を集めている。[[1]](https://www.krea.ai/)[[2]](https://huggingface.co/krea)[[3]](https://www.youtube.com/watch?v=0jssDQ9b9hI)

- **illustrious（Illustrious-XL）**: OnomaAI ResearchによるSDXLベースのイラスト/アニメ特化モデル。ネイティブ1536px高解像度対応で、Danbooruタグや絵師スタイル混ぜに強い。公式サイトやCivitaiなどで利用可能。[[4]](https://www.reddit.com/r/StableDiffusion/comments/1jjubsr/what_is_illustriousillustriousxl/)[[5]](https://huggingface.co/OnomaAIResearch/Illustrious-XL-v1.0)

- **Wan2.2**: Wan-Video/Alibabaのオープンソース動画生成モデル（MoEアーキテクチャ）。テキスト-to-ビデオ/画像-to-ビデオ対応、720p 24fps、映画的な制御が特徴。ローカル実行しやすく、ComfyUIワークフローで人気。[[6]](https://github.com/Wan-Video/Wan2.2)[[7]](https://docs.comfy.org/tutorials/video/wan/wan2_2)

- **Anima**: テキスト内での言及が中心で、広範な一般検索では具体的な公式情報が少なく、コミュニティ固有のファインチューンやニッチモデルである可能性が高い（詳細は確認できず）。

- **Qwen-Image**: AlibabaのQwenシリーズ関連の画像/編集モデルと推測されるが、具体的な「Qwen-Image」単体での詳細は検索で限定的。Qwen-VL系の高性能画像理解/生成機能が該当する文脈で使われている模様。

- **NovelAI**: 有名な有料アニメ特化画像生成サービス。v5などバージョンアップで背景描写などが強化されている。

- **その他（FLUX, LTXなど）**: FLUXはBlack Forest Labsの汎用モデルとして既知。LTXは動画モデルとしてWanと並んでローカル用途で言及される傾向。

これらの情報は2026年7月時点のウェブ検索に基づく。
