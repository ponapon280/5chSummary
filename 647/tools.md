# 元の本文
## 生成AI関連ツールレポート

提供されたテキストに基づき、言及されている生成AI関連ツールを分類して報告します。

### 画像・動画生成プラットフォームおよびGUI
#### ComfyUI (comfy)
- **概要**: ノードベースのGUI。画像・動画生成において幅広く利用されている。
- **選定理由**: 
    - ワークフローの構築および詳細な制御が非常に優秀であるため。
    - 最新モデル（HiDream-O1等）をノードとして組み込める拡張性が高いため。
    - 生成画像からプロンプトを適切に抽出できる機能があるため。
    - Gradio版と比較して歌詞を飛ばす確率が低く、実用的であるため。
- **話題**: バージョンアップによる速度向上や、LTX-2.3用の公式ワークフローの動作、インペイント機能の追加などが議論されている。

#### Forge neo (webUI reforge)
- **概要**: 画像生成用GUI。
- **話題**: マスクインペイントへの対応や、`supermerger`による階層マージの効果が言及されている。一方、インペイント時にマスク外の画像が微妙に変化する挙動が報告されている。

#### Stability Matrix
- **概要**: インストール管理ツール。
- **選定理由**: インストールの容易さから、クローン版等でエラーが出るユーザーへの推奨ツールとなっている。

#### EasyWan
- **概要**: ローカルAI環境。
- **話題**: プリセット機能が便利である一方、ComfyUIへの移行・設定は困難であるという対比で言及されている。

### クラウドGPU・生成サービス
#### RunPod
- **概要**: GPUレンタルクラウドサービス。
- **選定理由**: 
    - 高価なGPUをローカルに揃えるよりもコストを抑えられるため。
    - ローカルのワークフローをそのまま再現可能で、特に負荷の高い動画生成に有効であるため。
    - Paperspace等の代替手段として、LTX等の動画モデルを簡単に利用できるため。
- **注意点**: GPUの枯渇による利用しにくさや、ファイル操作の不便さ、サーバーのレイテンシ、課金リスクなどが指摘されている。

#### Seedance
- **概要**: 動画・音声生成サービス。
- **選定理由**: アニメ系のSDキャラを短尺アニメ化させるのに最適であるとされる。
- **注意点**: NSFW規制が厳しく、画像参照なしでの出力が困難である。

#### PixAI
- **概要**: 画像・動画生成サービス。
- **評価**: アダルト動画生成において成功している唯一のサービスとして挙げられている。

#### その他のサービス
- **HiDream / SenceNova**: 高速な生成速度（2kサイズを20秒程度）が評価されている。
- **higgsfield**: GPT Image 2を利用したNSFWコンテンツ生成が可能との話題がある。
- **Google colab**: 初心者が導入しやすく、利用し続けられている。

### 音声生成・LLMツール
#### Irodori-TTS (v3)
- **概要**: アニメ調の音声生成ツール。
- **選定理由**: 表情やナレーションなどの表現力が向上し、利便性が増したため（v3での制限撤廃や速度向上）。
- **話題**: 参照音声による調整や、アニメ顔画像からの声作成などの活用法が議論されている。

#### LM Studio
- **概要**: ローカルLLM実行環境。
- **話題**: `Sulphur2/10eros`専用のプロンプトエンハンサーが非常に優秀であると評価されている。

#### NotebookLM
- **概要**: Gemini連携のノートツール。
- **活用法**: 成功プロンプトや絵師タグの蓄積、Deep Researchによるキャラクター情報の把握など、プロンプト生成の補助として期待されている。

### 補助ツール・ユーティリティ
#### 開発・解析ツール
- **Codex / Claude Code**: ComfyUIのフォルダ構成解析、トラブルシューティング、個人でのツール作成に利用されている。
- **uv / venv / pip**: Python環境構築およびライブラリ導入に使用。
- **GitHub**: プロジェクト配布元として利用。

#### メタデータ・画像処理
- **xnconvert / remove-meta-alpha / 自作Webツール**: 複数枚の画像からのメタデータ削除や形式変換に使用。
- **imgbrd-grabber**: 学習素材集めのための高速画像収集に使用。

#### その他
- **MSI Afterburner**: グラボの電力制限（パワリミ）による発熱抑制とハードウェア保護に使用。
- **ChBrowser**: AI画像タグ表示機能を盛り込んだブラウザとして言及。
- **Vcode**: プロンプト管理ツールとして検討されている。
