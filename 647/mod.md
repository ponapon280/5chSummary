# 元の本文
## 生成AIモデルに関するレポート

提供されたテキストから判断すると、現在は**anima**が画像生成領域において非常に強い流行を見せており、動画生成領域では**Wan**や**10eros**（LTX-2.3ベース）への関心が高まっています。特にanimaは、学習の容易さと表現力の高さから、多くのユーザーがメインモデルとして移行・定着している傾向が見て取れます。

### 画像生成モデル

#### anima
- **選ばれている理由**
    - **圧倒的な学習効率**: LoRA作成が非常に高速（設定により20〜30分程度）で、試行錯誤が容易であるため。
    - **高い制御力**: Transformerベースであるためプロンプトへの反応が良く、衣服のボタン数などの詳細な指定や、Danbooruタグと自然言語の両方に対応している。
    - **動作の軽快さ**: 他のモデルと比較して軽量であり、導入ハードルが低い。
    - **表現の幅**: リアルな局部描写とアニメ顔を共存させることが可能で、「可能性の獣」と評されるほど汎用性が高い。
- **特徴・評価**
    - 階層マージによる詳細な調整が可能である。
    - インペイント性能が高く評価されている。
    - 課題として、背景を書き込みすぎる傾向や、特定の目の形の制御が難しい点が挙げられている。

#### illustrious (リアス)
- **特徴・評価**
    - 階層マージを重ねることで、非常に好みの絵柄（デジタル伴侶と称されるレベル）を追求できる。
    - 強度を上げても画像が壊れにくい安定性を持つ。

#### Z-Image / Z-Anime (ZIT, ZIE)
- **特徴・評価**
    - リアル系生成において非常に質が良いとされる。
    - 生成速度は速いが、複雑な絡みの描写（めり込み等）に弱く、版権キャラクターの再現性が低いという指摘がある。

#### Qwen-Image / Qwen Anime
- **特徴・評価**
    - 性能は高いが、Danbooruタグへの対応不足やLoRA作成の困難さがあり、ユーザーがanimaへ流出した要因となっている。

### 動画生成モデル

#### Wan (Wan2.1 / 2.2 / 2.6 / 2.7)
- **選ばれている理由**
    - 高いクオリティの動画生成が可能であり、特に男性器の形状再現や特定の性的な動き（騎乗位など）が向上しているため。
- **特徴・評価**
    - 実写系に向いており、バージョンによって評価が分かれる（2.6の方が評判が良いとの声あり）。
    - モデルサイズが非常に大きく、ストレージを圧迫する点が課題。

#### 10eros / Sulphur 2
- **選ばれている理由**
    - LTX-2.3をベースとしており、特に10erosはi2v（Image-to-Video）の強化モデルとして活用されている。
- **特徴・評価**
    - 非常に高いスペック（VRAM 12GB / RAM 80GB以上）を要求する。
    - 質感は良いが、顔が「バタ臭く（外人化）」なる傾向がある。

#### Seedance
- **特徴・評価**
    - アニメ系の短い動画生成に最適とされる。
    - 参照画像があれば高品質な描画が可能だが、未知の動きをさせようとすると劣化する。

### 言語モデル (LLM)

#### Gemma (gemma4B / 26B / 31B)
- **選ばれている理由**
    - 小説執筆や翻訳において有能であり、特に「常識改変」などの特殊なエロティックな文脈への理解度が非常に高いため。
- **特徴・評価**
    - モデルサイズに応じた使い分け（翻訳は4B、複雑な生成は31Bなど）が行われている。
    - 将来的にはスマートフォン内での動作が期待されている。

#### Gemini
- **用途**
    - キャラクターLoRA作成のための「3面図」生成に効率的である。
    - NotebookLMと連携し、個人の好みや設定を学習させたプロンプト生成に活用されている。

#### Grok
- **特徴・評価**
    - NSFW表現を学習済みで声や動きの質が良いが、規制が厳しいため出力に苦労する。一方で、画像生成用のプロンプト作成能力は非常に高い。

### 音声合成モデル

#### Irodori-TTS (v3)
- **特徴・評価**
    - 30秒の制限が撤廃され、短文生成が高速化した。
    - 新たなアノテーションの追加により、ナレーションと喘ぎ声などの共存が可能になり、表現力が向上している。
