**抽出されたモデル・ツール関連話題（除外対象を除く）**

### 1. Krea2（最も言及が多い）
- **主な話題**:
  - ベースモデル時点でアニメイラストの品質が安定して出やすい。
  - リアル系生成の増加が目立つ（特にエロ寄りのリアル）。
  - 背景の描き込みが強い。
  - 動画サムネ化して目を引く投稿が増えている。
- **選ばれている理由として挙げられている点**:
  - アニメイラストの再現性・安定感が良い。
  - リアル寄りの絵も比較的作りやすい。
  - ローカル環境でも扱いやすい（Forge Neoとの組み合わせ）。

### 2. Forge Neo / EasyForgeNeo
- **主な話題**:
  - アップデート後にVAE/text encoderの扱いが変わり、生成できなくなる不具合が発生。
  - 手動でVAEとtext encoderをコピーすることで復旧した事例あり。
  - Dynamic PromptsやWD14 Taggerとの相性問題が複数報告されている。
- **選ばれている理由**:
  - Anima以外のモデルでもローカル生成環境として広く使われている。
  - 拡張機能の対応状況が比較的早い。

### 3. LLM系（ChatGPT / Gemini / Claude / Codex / Fable）
- **主な話題**:
  - ChatGPT（特にCodex）は現在も実用性が高いという意見。
  - Geminiは日本語の自然な応答が強い。
  - Fableは高性能だがAPI料金が高く、石油王レベルでないと厳しい。
  - Claudeは使用制限が厳しいイメージが残っている。
- **選ばれている理由**:
  - 自然言語での指示出しやすさ。
  - 日本語の理解度・出力の自然さ。
  - コストパフォーマンス（Codexが比較的評価が高い）。

### 4. その他
- **Ideogram**: 日本語の理解度が非常に高いと評価された事例あり。
- **ComfyUI**: バージョンアップ（v0.27.0）でint8モデルサポートが追加された。Krea2関連のノード（ModelMergeKrea2）も登場。
- **uni-1**: SFWであれば小説から漫画生成が可能という情報あり。
