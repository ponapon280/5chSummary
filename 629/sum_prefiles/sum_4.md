# なんJ(5ch) AI画像生成スレッド レポート (レス649-848)

## 1. スレッド概要
- **対象期間**: レス649〜848 (約200レス)。
- **勢い**: 以前の300レス/hから168レス/hへ低下（>>671, >>677）。5chドメイン変更やスレスパ2人気による影響か。書き込み環境改善（mate/mate対応 >>673, >>681）で回復傾向。
- **主要テーマ**: AI画像/動画生成ツール（ComfyUI中心）のTips共有。新モデル「Anima preview2」リリースが最大の話題。アップスケーラー、LoRA作成、プロンプト技法、TTSが活発。エロ画像生成の再現性向上やハードウェアTips多め。スレスパ2（ゲーム）の話題が横流し。

## 2. 主要トピックと議論ポイント

### 2.1 Anima Preview2 リリースと評価 (>>744〜, >>757, >>819など最多話題)
- **アップデート内容** (>>749公式引用): 小幅アップグレード。中解像度トレーニング延長、自然言語理解向上、正則化データセット導入。1024解像度超で崩壊注意。ベースモデルで美的調整なし。
- **改善点**:
  - 線くっきり、ディティール向上、アートスタイル安定、キャラ知識増加 (>>757, >>762)。
  - 絵師タグ反応微調整（本来の絵に近づくが雑絵師で品質低下 >>819）。
  - 難構図（POVバックハグ、聴診器おっぱい >>833）容易。Regional prompting精度高 (>>832)。
- **不満点**:
  - 劇的変化なし（>>752, >>753）。旧LoRA互換性低下（>>758）。
  - 1024解像度破綻残存、高解像度未対応（>>818）。絵師タグ効き変動（>>791）。
- **活用法**:
  - 下書き生成→リアスi2i/hires (>>771, >>814, >>838)。
  - LoRA作成待機派多数（>>759, >>772, >>813）。自作LoRA100人分再学習ダルい（>>772）。
  - Civitai/HFで公式ファイル確認（SHA256一致 >>846）。

### 2.2 アップスケーラー/高速化ツール
- **RTX VSR/FlashVSR** (>>653, >>674, >>724):
  - ComfyUI_NVIDIA_RTX_NodesインストールTips: pip install nvidia-vfx (issues解決 >>674)。
  - リアルタイム再生向け軽量。動画前処理でブラー減、Rife TensorRTと併用推奨（>>724）。RAM食う（>>737）。
- **Real ESRGAN** (>>659, >>672): 6倍過多推奨せず4x UltraSharp。
- **A1111/Forge Neo対応** (>>727): nvidia-vfx活用。Neoは拡張一部非対応（>>785）。
- **その他**: LTX2.3動画生成（RTX5080+128GBで15秒56分 >>731）。distilled LoRA併用（>>817）。

| ツール | 利点 | 注意点 |
|--------|------|--------|
| RTX VSR | 高速、低解像度ブラー減 | エンコード品質低、RAM多 |
| Real ESRGAN | 高品質 | 倍率過多で破綻 |
| TensorRT | 動画高速 | 比較でRTX VSR遅め（>>678） |
