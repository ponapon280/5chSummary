### 生成AIモデルに関する抽出話題（除外モデル除く）

ログ全体から、除外モデル（NovelAI/NAI, Pony, illustrious/リアス/ill/IL, Noobai/Noob, FLUX, Wan, Qwen）を除いた「モデル」関連の言及を抽出。主にStable Diffusion系や動画生成特化モデル（I2V/V2V系）が対象。ツール（ComfyUI, A1111, Forgeなど）や一般論（Stable Diffusion）はモデル名として扱わず除外。選ばれている理由や文脈を明記。

#### 1. **Z-image / Z-Image Base / zimage**
   - **言及箇所**: 59, 96, 134, 137
   - **話題概要**:
     | レス | 内容・理由 |
     |------|-------------|
     | 59 | イラストリアス（除外）から離れたいが、Z-Imageの上位生成環境で同等に便利なモデルが出ておらず移行しづらい。 |
     | 96 | 今年の画像生成はZ-imageの年になりそう。Baseが出たら誰かがリアス/Noob（除外）並みにファインチューニング（FT）する可能性。edit機能も年内に完成予想。 |
     | 134 | Baseが出てしばらくしたら誰かが学習開始するかも。それで人気わかる（やらなかったら現状維持）。 |
     | 137 | 英語漫画生成ならzimageも悪くない（エロ漫画には不向き？）。 |
   - **選定理由のポイント**: 上位生成環境の利便性が高く、FT次第でメジャー化の期待大。英語漫画向き。

#### 2. **SDXL**
   - **言及箇所**: 87
   - **話題概要**: VRAM 6GBでSDXLを使っている事例。起動オプションでFP8指定+Tiled VAEでVRAM大幅削減。
   - **選定理由のポイント**: 低VRAM環境でも動作可能（最適化次第）。

#### 3. **Lumina**
   - **言及箇所**: 87
   - **話題概要**: VRAM 8GBでLuminaを動かす事例。GGUF使用。
   - **選定理由のポイント**: 低VRAM（8GB）で高解像度動作可能。

#### 4. **QI2512 / 2512**
   - **言及箇所**: 108, 112
   - **話題概要**:
     | レス | 内容・理由 |
     |------|-------------|
     | 108 | 2512は日本風アニメイラストが苦手で写実寄りの絵柄になる。 |
     | 112 | QI2512は重すぎてFTできないと叩かれ、開き直って世界理解（不細工アジア人再現）の精度に舵を切ったと思われる。 |
   - **選定理由のポイント**: 写実・世界理解精度が高いが、日本風アニメ不向き、重くてFTしづらい。

#### 5. **EXITIUM VICTRIX（ブチギレモデル）**
   - **言及箇所**: 173, 182, 185
   - **話題概要**: SVIワークフローでHighモデルとしてSmoothmixと比較。動きが抑制されがちなSmoothmixに対し、よく動く（固まりにくい）。ただし精度低め（プロンプト外の虹が出現など）。暴れさせたい時に運用。3段生成（ブチギレH 1step → dasiwaH 3step → dasiwaL 3step）でキャラ崩壊防ぎに有効。
   - **選定理由のポイント**: 動きのダイナミックさ（固まりにくく暴れやすい）が強み。Smoothmixの常用代替として使い分け。

#### 6. **smoothmix / Smoothmix**
   - **言及箇所**: 99, 106, 138, 173, 177
   - **話題概要**: 動画生成（SVI/I2V）でHigh/Lowモデルとして使用。Light2Xマージ済み推奨。EXITIUM VICTRIXより精度高く安定（常用）。steps数調整で改善。dasiwaと組み合わせで日本人風出力。
   - **選定理由のポイント**: 安定性・精度が高く常用向き。暴れさせたい時はEXITIUM VICTRIXに置き換え。

#### 7. **DaSiWa / dasiwa / dasiwaH / dasiwaL / Dashiwa**
   - **言及箇所**: 99, 106, 138, 173, 177, 182
   - **話題概要**: Lowモデルとして「安定のDaSiWa固定」。SVIでHigh（smoothmix/EXITIUM）と組み合わせ。3段生成でキャラ崩壊防ぎ（dasiwaH 3step → dasiwaL 3step）。smoothmix+steps調整で成功事例。
   - **選定理由のポイント**: 安定性抜群（Low固定推奨）。キャラ崩壊防ぎに特化、Highモデルとの相性良し。

#### 8. **Light2X / Light2XLora**
   - **言及箇所**: 106
   - **話題概要**: Light2Xマージ済み（dasiwa/smoothmix）かLoRA挿入推奨。steps数増やして解決。
   - **選定理由のポイント**: 動画生成の高速化・安定化に寄与。

#### 9. **Pusa / Phantom**
   - **言及箇所**: 229
   - **話題概要**: SVIにPusaとPhantomを入れると動きが良くなった気がする。
   - **選定理由のポイント**: 動きの向上に効果的（主観）。

#### 抽出ノート
- **全体傾向**: 動画生成（SVI, I2V, PainterLongVideoなど）関連でモデル議論が集中。静止画はZ-image/SDXL/Luminaが低スペック環境向きとして挙がる。選定理由は主に「安定性」「動きの良さ」「低VRAM動作」「精度/FTしやすさ」。
- **除外適用**: Wan2_2-I2V（Wan関連除外）、Qwen3-VL（除外）、Noob/Illustrious/リアス/NAI/FLUX（複数除外）。2512はQI2512の略称とみなし抽出。
- **不明/マイナーモデル**: 上記以外に明確なモデル名言及なし（例: ブチギレHはEXITIUMの愛称）。追加抽出対象なし。
