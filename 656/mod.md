# 元の本文
**生成AIモデル動向レポート**

### 流行モデルのまとめ
本テキスト全体を通じて、**Anima（WAI-Anima / Anima base）**が圧倒的に話題の中心となっており、現時点での主流モデルである。Illustrious（リアス）はAnima登場以前の主流として比較対象として頻出するが、徐々に主役の座を譲りつつある状況が読み取れる。Gemma4はLLMとしての言及が散見されるが、画像生成モデルとしての位置づけではない。

### Anima（WAI-Anima / Anima base）の台頭
Animaは複数のログで「最も言及されているモデル」として突出しており、**リアスからの移行が進んでいる**ことが確認できる。

**選ばれている主な理由**:
- 自然言語プロンプトの理解力が高く、指示通りの出力が得られやすい
- エロ表現への耐性・出力品質が優れている
- 複数キャラのポーズ・絡み表現がリアスより強い
- base版は自由度が高く、LoRA使用時の柔軟性に優れる
- 低VRAM（8GB）環境でも実用的な生成が可能

一方で、派生モデル（WAI-Animaなど）は安定感が増す一方で「baseより突出した良作が出にくい」という声もあり、用途によってbaseとFTモデルの使い分けが議論されている。

### Illustrious（リアス）からの変化
リアスはAnima登場以前の主流モデルとして繰り返し言及されるが、**状況が明確に変化**している。

- Animaとの比較で「プロンプト理解力・キャラ制御」で劣るとの評価が増えている
- 学習データの元として使われるケースは残っているが、新規採用は減少傾向
- 「リアス時代は…」という形で過去のモデルとして語られることが多く、移行完了のフェーズに入っている

### その他の注目点
- **Gemma4**: LLMとしてQwenの代替候補として名前が上がる。日本語性能と検閲の緩さが評価されているが、画像生成モデルではないため本レポートの主眼からは外れる。
- その他（Cotton、Seedance 2.0、Ideogram 4.0など）は散発的な言及に留まり、Animaほどの勢いは見られない。

**結論**: 今回のログでは、**Animaがリアスを事実上置き換えつつある**という変化が最も顕著なポイントである。リアス由来の学習資産を活かしつつ、Anima baseを新たなベースとして定着させようとする動きがコミュニティの主流となっている。
