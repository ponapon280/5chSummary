# なんJ(5ch) AI画像生成スレッド ログレポート

## 概要
- **対象ログ**: レス番号23〜246の会話ログ（主にAI画像生成ツール、特にAnima、WAI-Illustrious（以下WAI）、Chenkin NoobなどのSDXL/Animaモデルに関する議論）。
- **主なテーマ**: Animaのプレビュー版（v3-preview）の評価・課題、SDXLモデル（WAIなど）との比較、ComfyUIの挙動、LoRA学習、LLM（Gemma4、Qwen3.5など）活用、プロンプト構築、ControlNet（CN）不足、動画生成の限界など。ユーザー層は中上級者（ComfyUI/Forge Neoユーザー中心）で、赤ちゃんユーザー（初心者）も混在。
- **全体傾向**: Animaのポテンシャルを高く評価する一方、制御の難しさ（画風不安定、多人数描写の優位性）とSDXLからの移行タイミングで意見分かれる。LLMをプロンプト補助に活用するTips共有が活発。エロ画像生成（ガニ股、排泄描写）が頻出。

## 主要トピックと議論まとめ

### 1. Animaの評価・課題
- **高評価点**:
  - クオリティの高さ（頭一つ抜けた性能、自然言語対応、多人数描写安定）。NAIより優位（>>226）。
  - 自作LoRA作成可能（>>223,224）。画風再現で調整次第でワンランク上（>>192）。
  - プロンプト柔軟性（自然言語+タグ混合）。将来性あり（>>152,156）。
- **課題・不満**:
  - **画風/一貫性不安定**: アーティストタグ混ざりにくい（>>39）、キャラタグで原作寄りになる（>>49）、暴れやすい（>>165）。ゴミ絵が出やすい（>>164）。
  - **ポーズ制御難**: ガニ股（bowlegged pose）出にくい（>>79,92,100）。解決策: standing+(bowlegged pose:1.5)、knees apart feet together弱め（>>102,209）。逆立ち化回避に解像度/部位指定（>>134）。
  - **自然言語の壁**: DeepL翻訳不十分（>>175）。LLM（Qwen3.5推奨）で補完必須（>>185,186）。
  - **CN未対応**: 高速化/VRAM削減待ち（>>78,238,240）。SDXL（WAI）で代用推奨（>>177）。
  - **その他**: 疑似排泄微妙（>>162）、チャラ男介入（>>210）。
- **移行タイミング**: SDXL（WAI）が「オールマイティで十分」（>>55,57,150）。Animaはプレビュー段階でLoRA/CN充実待ち（>>157,169）。「使いこなせないから使わない」（>>168,172）。

| モデル | 強み | 弱み | ユーザー所感 |
|--------|------|------|-------------|
| **Anima v3-preview** | 高クオリティ、自然言語、多人数 | 制御難、画風不安定、CNなし | ポテンシャル高/乗り換え微妙 |
| **WAI-Illustrious** | 安定、無難万能 | 暗闇弱、キャラ中央寄り | 結論モデル/冒険不要 |
| **Chenkin Noob (最新)** | 版権キャラ追加、無印Noob進化 | 不安定、暴れ馬、高速化LoRAで低下（>>220） | 位置不明/WAIに戻る |
| **NAI** | - | 多人数崩壊、アナルカンスト | サブスク継続中/新モデル待ち |

### 3. LLM活用（プロンプト生成）
- **人気モデル**:
  - Qwen3.5-35B-A3B-Uncensored-HauhauCS-Aggressive (Q5推奨 >>189,190)。Animaと相性良（>>185）。
  - Gemma4 26B A4B/Q4_K_M（>>64,184）。
- **Tips**:
  - Thinking停止: {% raw %}{%- set enable_thinking = false %}{% endraw %} (LM Studio >>194,201)。
  - シスプロ: Animaページ貼付+「混合形式・推奨」「masterpiece省略」（>>191,196,202,236）。同一出力化成功（>>211）。

- **ローカルLLM問題**: コンテキスト長超えで記憶なし（>>62,111）。

- **共有**: Adetailer乳首LoRA（Anima-preview3 >>119）。世界樹ニキの調整法（強度/ステップ総当り >>222）。

## ユーザー傾向・推奨
- **中上級者**: Anima実験中だがWAI安定派多数。LLM+シスプロでAnima制御。
- **赤ちゃん**: Forge Neo/テンプレ推奨、WF共有待ち。
- **全体推奨**:
  - Anima入門: Qwen3.5シスプロ+自然言語混合。
  - 安定派: WAI v16+LoRA。
  - 環境: 12GB VRAM以上（Anima+LLM >>246）、RAM128GBでMoE活用（>>231,243）。

## 結論・洞察
スレはAnimaの「高性能だが制御難」を巡る議論が中心。SDXL（WAI）が「完成形」として君臨しつつ、Animaの自然言語/CN進化に期待。コミュニティ活性化の鍵はWF/シスプロ共有（建設的レス歓迎 >>94）。正式版/CNリリースで移行加速か。エロ特化Tips多めで、なんJらしい実践志向。次ログでAnima v3正式/CN対応の反応注目。
