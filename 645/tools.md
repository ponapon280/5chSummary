## 生成AI関連ツール調査レポート

提供されたテキストに基づき、生成AIに関連するツールをカテゴリー別に分類し、その特徴や選定理由についてまとめます。

### 画像・動画生成ツール
#### ComfyUI (comfy)
- 特徴: ワークフロー（WF）ファイルをドラッグ＆ドロップで読み込める点や、Forge Neoよりも画質が良いと感じるユーザーがいる。
- 課題: 起動直後の画面が「黒い方眼紙」のようで、自動保存機能がないため不便に感じる場合がある。また、導入難易度が高く、特定のモデル動作時にエラーが発生することが報告されている。

#### WebUI / Forge Neo / easy reforge
- Forge Neo: easy reforgeの代替案として推奨されており、ComfyUIよりも心地よく利用できるとの意見がある。
- Forge Couple: Animaにおける領域書き分けによる複数キャラの配置に便利とされる。
- A1111: 生成画像が自動的にフォルダに保存される仕様が評価されている。

#### Seedance 2.0
- 選定理由: 他ツール（GrokやWan）より性能が高く、服のボタンを一つずつ外すような細かいモーションや、音声参照機能による特定セリフの喋らせることが可能であるため。
- デメリット: 従量課金が高額である。

#### その他画像・動画ツール
- Qwen Image Edit (QIE): エロコンテンツを弾かない点と、背景合成に便利な点が評価されている。
- SVI PRO: 長尺の動画生成において現時点でも有効なツールとされている。
- nano-banana (pro): 画像の水増しに利用されており、90年代風の画風が出やすい特性がある。
- Stability Matrix: モデル管理機能が便利であるため愛用者が多い。
- Krita: Animaに対応したため試行されている。
- tensor: Anima LoRAの作成に利用されている。

### 学習・タグ管理・補助ツール
#### 学習・キャプションツール
- Anima LoRA Factory: 操作が簡単で、手軽にLoRAが作成できるため利用されている。
- BooruDatasetTagManager: タグ付けと確認が一貫して行え、一括修正・削除・追加が可能なため便利とされる。
- TagGUI: キャプション付け（タグ付け）を非常に楽に行えるため推奨されている。
- Kohya (kohya_ss / Kohya_lora_param_gui): 簡易インストーラーにより導入が容易である。
- sd-scripts: トレーニングに使用されるが、VRAM不足やキャプションファイルの有無による不具合が報告されている。

#### プロンプト・ワークフロー補助
- 自作プロンプト生成ツール: 初心者がシチュエーションをランダムに生成し、コピーして利用するのに有用。
- Prompt Relay: タイムライン記述とスライダー調整が可能であるため活用されている。
- Power Lora Loader: LoRA Managerで表示されないLoRAでも正常に読み込み・適用できる。
- Google Translate Text Node: 自然言語が苦手な場合に、品質タグ等を英訳して結合するために利用される。
- Tagger: LLMと比較して「素直」であると評されている。

### コーディング・開発ツール
#### Codex
- 選定理由: アプリ実装の品質が高く、現状で「一強」と評価されており、タスクが長引いても性能が落ちにくいとされる。
- 課題: PC内部で直接作業し逐一許可を求める仕様が不便と感じるユーザーもいる。

#### Claude / Claude Code
- 選定理由: 言語化能力が高く、曖昧な要求でも的確なアウトプットが出ること、またComfyUIのワークフロー構築における信頼性が高いため。

#### Cursor
- 特徴: AIによる高速開発（バイブコーディング）に利用されるが、小規模開発以外での利用には危険性が伴うとの警鐘がある。

#### その他開発ツール
- Devin: 長時間の自律的な丸投げ作業に向いている。
- Android Studio (Gemini): 古いアプリの改修に有効に機能した事例がある。
- browser-use: ブラウザ操作を可能にするためのプラグインとして利用される。
- Roo Code / LM Studio: 組み合わせて開発に利用されている。

### LLM（大規模言語モデル）
#### Qwenシリーズ
- 特徴: ローカル動作が可能。標準版は検閲があるが、Uncensored版を使用することで制限なく生成できる。

#### その他のLLM
- ChatGPT: 「クイック回答」による辞書的な高速利用や、素材探しの代替手段として利用されている。
- Gemini: 学習データのパターン提示などのアドバイスに利用される。
- Deepseek: Geminiからの乗り換え先として利用されている。
- 天秤AI: 無料枠でClaudeなどの制限を避けて試すための入り口として推奨されている。
- GMO: エロ文書作成などの用途で利用価値があるとされている。

### 音声生成ツール
#### Irodori-TTS
- 選定理由: 小説を音声化する際、違和感のない読み上げが可能であるため高く評価されている。
