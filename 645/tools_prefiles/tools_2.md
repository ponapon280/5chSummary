## 生成AI関連ツール抽出結果

### 画像生成・動画生成ツール
#### WebUI
- 個人でバイブコーディング（AIによる高速開発）して作成することが遠くないと言及されている。

#### ComfyUI (comfy)
- アップデート時に未知のノードが大量に表示される挙動について言及がある。
- 初心者がAnimaでアーティファクト生成に時間を費やしたというエピソードがある。

#### Seedance 2.0
- ガチエロは無理だが、ややエロ（乳首まで）の動画生成が可能。
- 服のボタンを一つずつ外すモーションに成功しており、他ツール（GrokやWan）より性能が高いと感じられている。
- 音声参照機能があり、推しキャラに特定のセリフを喋らせることが可能。
- 従量課金が高額である点（1080p 10秒で約$1.40）がデメリットとして挙げられている。

#### Runway
- 微エロ動画の作成に使用されていたが、垢BANされた例が報告されており、年間プランは避けるべきとの助言が出ている。

### コーディング・エージェントツール
#### Codex
- 使用制限が少なく、性能が良い。
- SNSで絶賛されており、ダウンロード数が急増している。
- タスクが長引いてもアホになりにくいと感じるユーザーがいる。

#### Claude Code
- 一晩中動かしてデータセットやパイプラインを構築して利用しているユーザーがいる。

#### Devin
- 長時間の丸投げ作業（自律的エージェント）に向いているツールとして言及されている。

### その他AI活用ツール・カスタムノード
#### バイブコーディング（手法/ツール群）
- プログラムを詳しく書けない人でも圧倒的に早くツールを作成できる点が高く評価されている。
- 小規模なツール作成には向いているが、仕様が複雑になるとAIが修正できなくなる限界がある。

#### 自作画像分類ツール（ChatGPT作成）
- キャラクターやシチュエーションタグでフォルダ分けができる利便性が挙げられている。

#### Comfy Couple (カスタムノード)
- 特定の構図やキャラクターの制御に使用される。
- Claudeを用いて改造し、自分好みの挙動に調整して利用している例がある。

#### Google Translate Text Node (ComfyUIノード)
- 自然言語が苦手な場合に、品質タグや部族語（プロンプト語）を英訳させて結合するために利用されている。

#### LLMノード (ComfyUI等)
- Anima等のモデルで、単調な構図を避けるために自然言語をランダムに生成してプロンプトに組み込む用途で推奨されている。
