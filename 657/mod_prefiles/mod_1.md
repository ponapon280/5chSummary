### Anima / WAI-ANIMA（最も言及が多い）
- **主な話題**:
 - SDXLからAnima（特にWAI-ANIMA / base1.0 / hakushi / cottonなど派生）への移行が活発。
 - 自然言語プロンプト（日本語含む）が効きやすい。
 - 構図出し・レイアウトの安定性が高い。
 - Turbo LoRA使用時の絵柄変化・安定性。
 - anytestやControlNetとの相性、LoRA学習時の解像度（512 vs 1024）の違い。
 - ネガティブプロンプトや強度タグ（( ) :2 など）の効き方。
- **選ばれている理由**（ログ内）:
 - 「自然言語で思い通りの構図が出やすい」「SDXLよりマシ」「Turboで安定する」「XLでやる意味がなくなった」。
 - 絵師タグやキャラLoRAとの相性が良く、自由度と安定性のバランスが良いと評価。
### Ideogram 4.0 / Ideogram4
- **主な話題**:
 - JSONプロンプト / bounding boxでのレイアウト制御。
 - テキスト生成能力（特に日本語・漢字）。
 - 検閲の強さ（エロは弱い、乳首以外はSD1.5並み）。
 - ローカル版 vs API版の画質差（Turbo/標準/Qualityの違い）。
 - Image参照機能、Prompt Builder KJノードの使い勝手。
- **選ばれている理由**（ログ内）:
 - 「bounding boxで配置できるのが凄い」「ローカルでも初代SD並みかそれ以上のことが出来る」「テキストが強い」。
 - ただしエロ用途では「学習不足で残念」との声も。
### リアス（Illustrious / ill / IL）
- **主な話題**:
 - 以前の主力モデルとして言及（Animaに移行した人が多い）。
 - Animaとの絵柄再現比較（同じ絵柄を出すのが難しい）。
- **選ばれている理由**（過去）:
 - 大型ファインチューン済みで絵柄が安定していたが、現在はAnimaに取って代わられつつある。
### LTX（LTX-2.3 / ltx2.3 / 10Eros）
- **主な話題**:
 - Ideogram4.0で作った静止画をベースに動画生成。
 - ControlNet（Depth）との組み合わせ。
 - 声のプロンプトの難しさ。
- **選ばれている理由**:
 - 「Ideogramで可愛いキャラを作ってLTXで動画にできる」点が評価。

**抽出されたモデル関連話題（除外対象を除く）**
### 1. Ideogram（Ideogram4 / Ideogram4.0）
ログ内で最も多く言及されている非除外モデル。
**主な話題・評価**
- **テキスト描写・指示の強さ**: 文字を綺麗に出せる点が特に評価されている。「fp16版が欲しい」「テキスト表示が死んでるfp8は微妙」といった声あり。
- **構図制御の優位性**: bounding box形式やJSONプロンプトで配置を指定できる点を「凄すぎる」「革命的」と高評価。Couplesなどの複雑な配置で特に有用とされる。
- **手書き指示の対応**: 手書きラフや指示書を分析してJSONに変換する使い方が提案されている。
- **検閲・エロ表現**: エロは学習が弱く（おっぱい・乳首以外はSD1.5並）、検閲も厳しめ。ただし「正しくJSONプロンプトで生成してないだけ」という指摘もあり、回避方法としてJSON形式の使用が推奨されている。
- **API版 vs ローカル版**: API版の方が画質が良いという声があり、Turbo/標準/Qualityの3段階がある。ローカル版でも「初代ぐらいのことはできる」との評価。
- **用途の提案**: プロンプトビルダー（KJノード）を使ったワークフロー構築、画像参照機能の有無についての議論。
**選ばれている理由（ログ内）**
- 構図指定のしやすさ（特にJSON/bounding box）
- テキストの安定感
- ローカルでも一定水準のクオリティが出せる点
### 2. Gemma / Gemma4 12B（LLM）
画像生成モデルではないが、生成補助として言及あり。
**主な話題**
- エロチャット用途でローカル運用を推奨（データセンター負荷軽減）。
- Googleがオンデバイス指向である点が特徴として挙げられている。
- 画像読み込みは「ガバガバ」との評価。
**選ばれている理由**
- ローカルでエロチャットができる軽さ・速度

- WAI（WAI-ANIMAのWAI部分）についてはAnima除外の関係でモデル単体としての言及は薄く、baseとの違いを議論する程度にとどまっていたため除外。
- その他に「cotton」「hakushi」「kimi」などの名称が出てきたが、いずれもAnima系派生モデルと見なされるため除外対象。
