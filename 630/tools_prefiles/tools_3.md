#### 理由が明記されたツール選択例
- **ComfyUI**: EasyWan卒業推奨（更新停止、最新モデル/動画生成対応）。VRAM効率良い環境構築可能だが学習曲線急（WF理解必要）。Stability Matrixで簡単インストール。
- **Stability Matrix**: 一覧インストールで楽（ComfyUI/EasyWan対応）。
- **--gpu-only**: 5090の32GB VRAMフル活用で高速化（画像生成2.5秒、動画は非対応）。
- **kohya sd-script GUI**: 最新モデル対応、Windows版あり（kohya_ss GUIは非対応の代替）。
