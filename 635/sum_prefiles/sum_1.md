## 1. スレッド概要
- **テーマ**: AI画像/動画生成ツール（Stable Diffusion系、ComfyUI WF、LoRA）、LLM（Gemma4など）のローカル運用、モデル比較。主に二次元エロ/アニメ生成に特化。
- **全体の流れ**: 画像共有（Anima/Ilias生成例）から始まり、モデル進化議論 → 動画生成トラブルシュート → LLM性能比較 → ComfyUI初心者支援 → ストレージ/スペック問題へ。活発な情報交換と画像/動画共有が特徴。Animaの高評価とeasywan離脱推奨が目立つ。
- **参加者傾向**: 実践者中心。初心者質問多めで上級者がアドバイス。エロ生成（母乳、フェラ、騎士姫など）実例共有。

## 2. 主要トピックと議論まとめ

### 2.1 AI画像生成モデル比較
- **Animaの評価爆上がり**:
  - 高解像度（1024→1536x1536予測）、構図力/LoRA適性◎、自然言語プロンプト理解力高（例: 鞄紐自然、乳首吸い構図完璧、キャラ名+版権で再現）。
  - Ilias（リアス）超え論争: 構図/Lora/高解像度で既に上回る（CN追加で決定的）。ただTE（テキストエンコーダ）弱く奇形リスクあり（部族語併用推奨）。
  - 例: >>75,116,129,132,138,143,220。キャラ再現（aris maid blue archive）でトークン効率良。
- **Ilias/hakushi**: 線質/二次エース。Anima待ちつつ現役（>>36,38,43）。
- **その他**: GPTImage2（微エロ限界、Arena一瞬登場）、Google Gemma系。LoRA作り直し覚悟（preview1→2無駄化）。

| モデル | 強み | 弱み | 予測 |
|--------|------|------|------|
| Anima | 構図/高解像/プロンプト理解 | TE弱、CNなし | 1536版で次世代 |
| Ilias | 線質/LoRA | 解像度/構図 | CNで衰退？ |
| hakushi | メリハリ線 | - | 二次エース継続 |

### 2.2 動画生成ツール/トラブル
- **easyWan2.2/i2v問題**:
  - 母乳LoRA+乳揺れでイソギンチャク化（>>56）。ベースモデル非エロ学習が原因→モデル変更推奨。
  - WF前提がfastmix固定→サブグラフ修正要（>>86）。SmoothMix/dasiwa WF移行連呼（>>87,91,93,96,97,118）。
  - 初心者ルート批判: easyReforge→easyWanの罠。StabilityMatrix/ComfyUIテンプレ推奨（>>111,114,119,122）。
- **LTX2.3/NSFW MMAUDIO**:
  - LTX: start-middle-end動画共有（>>58）。MMAUDIO: ジュポ音/クチュ音得意（24fps対応、動画内容検知？ >>66,71,79）。
  - VoiceDesign v2対応カスタムノード共有（>>112,140,153,155,170-174）。
- **その他**: For Loopメモリリーク（長尺アプスケ失敗 >>128,137,231）。実写化例（>>202）。

**移行推奨フロー**:
```
easyWan → SmoothMix v1 High/Low (LoRAオフ+プロンプトで母乳可 >>118)
       → dasiwa WF (高画質/高速 >>98)
       → ComfyUIテンプレ/Portable/StabilityMatrix
```

### 2.3 LLMローカル運用（Gemma4中心）
- **Gemma4高評価**:
  - 26B A4B/MoE軽快（5070ti Q8:20t/s >>82）。日本語自然/エロ強（Qwen3.5超え >>104,151,161,167,177）。
  - LM Studioロード失敗多発（>>67,200,203,204）。更新/コンテキスト長/GPUオフロード調整で解決。
  - heretic版エロ特化（>>74）。Q4-Q8推奨（64GB RAM+16GB VRAMで26B Q6_K可）。
  - 比較: Qwen3.5重/不自然日本語で即削除勢多（>>150,158,189）。
- **運用Tips**: temp=1.0推奨（>>85）。システムプロンプトで人格/思考モード設定。
- **課題**: ストレージ爆食（1TB/月 >>149,156）。画像生成暴走（>>187,205）。

| LLM | スペック例 | 強み | 弱み |

| Qwen3.5 27B/35B | - | 英語？ | 重/不自然JP |
